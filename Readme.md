# Личный проект «Кекстаграм»

Веб-приложение для загрузки и просмотра фотографий. Приложение позволяет пользователям загружать изображения, применять к ним эффекты, добавлять хэштеги и комментарии, а также просматривать фотографии других пользователей.

## Описание проекта

Кекстаграм позволяет пользователям:
- Загружать свои фотографии
- Применять к изображениям различные эффекты
- Масштабировать изображения перед загрузкой
- Добавлять хэштеги и комментарии
- Просматривать фотографии других пользователей
- Фильтровать фотографии по различным критериям

## Технологии

- **HTML5** — семантическая структура страницы
- **CSS3** — стилизация интерфейса
- **JavaScript (ES6+)** — основная логика приложения
- **Модульная архитектура** — код разделен на независимые модули

### Библиотеки

- **Pristine** — валидация формы (хэштеги, комментарии)
- **noUiSlider** — слайдер для регулировки интенсивности эффектов

### Работа с сервером

- **Fetch API** — асинхронные запросы к серверу
- REST API для получения и отправки данных
